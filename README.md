# Gerenciador de Tarefas

Este projeto é um **Gerenciador de Tarefas** simples desenvolvido em **React**. Ele permite adicionar tarefas com descrições, marcar como concluídas e excluir tarefas. As informações são armazenadas localmente no navegador.

## 📌 Funcionalidades

- Adicionar tarefas com descrição
- Marcar tarefas como concluídas
- Excluir tarefas
- Armazenamento local para persistência dos dados

## 🛠️ Tecnologias Utilizadas

- **React**
- **CSS** para estilização
- **Local Storage** para salvar as tarefas
